# Thunderbird Puppet Module for Boxen

Install [Thunderbird](https://www.mozilla.org/en-US/thunderbird/), a free email application.

## Usage

```puppet
include thunderbird
```

## Required Puppet Modules

* `boxen`

## Development

Write code. Run one of the following tasks to test it:
* `rake syntax`
* `rake style`
* `rake spec`
* `rake build`
